## Overview

Tools are discovered **implicitly** based on the connections listed in the current scope's Virtual MCP. There is no explicit `tool_search` built-in tool — available tools are automatically provided from the connections configured in the active scope.

## How Discovery Works

When you enter a scope (organization, project, or agent), the tools available from all connections in that scope's Virtual MCP are automatically listed. You can then enable specific tools with `tool_enable` before invoking them.

<Callout type="info">
  Tool discovery is automatic. You don't need to search for tools — they're available based on your current scope's Virtual MCP configuration.
</Callout>
