## Visão Geral

Ferramentas são descobertas **implicitamente** com base nas conexões listadas no MCP Virtual do escopo atual. Não existe uma ferramenta `tool_search` integrada — as ferramentas disponíveis são automaticamente fornecidas a partir das conexões configuradas no escopo ativo.

## Como Funciona a Descoberta

Quando você entra em um escopo (organização, projeto ou agente), as ferramentas disponíveis de todas as conexões no MCP Virtual daquele escopo são automaticamente listadas. Você pode então habilitar ferramentas específicas com `tool_enable` antes de invocá-las.

<Callout type="info">
  A descoberta de ferramentas é automática. Você não precisa pesquisar ferramentas — elas estão disponíveis com base na configuração do MCP Virtual do seu escopo atual.
</Callout>
