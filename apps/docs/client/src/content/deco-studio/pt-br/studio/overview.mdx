## Control plane open-source para seus agentes de IA.

deco Studio é onde você contrata agentes, conecta ferramentas, organiza tudo em projetos e acompanha cada token e cada real. Self-host ou use a nuvem. Construído sobre o [Model Context Protocol](https://modelcontextprotocol.io).

### Agentes

Navegue e contrate agentes de IA na loja integrada. Cada agente vem com skills específicas e prompts testados. Agentes se compõem — podem chamar uns aos outros. Toda ação é rastreada com atribuição de custos.

### Conexões

Configure conexões MCP pela interface web com OAuth em um clique. Monitore latência, erros e custos por conexão. Compartilhe ferramentas com sua equipe sem compartilhar credenciais.


### Projetos

Projetos agrupam agentes e conexões em torno de um objetivo. A UI do projeto se adapta ao que está dentro — adicione um agente de conteúdo e uma conexão CMS, a sidebar mostra gestão de conteúdo; adicione um agente de analytics e um banco de dados, mostra dashboards e queries.

### Observabilidade

Gastos com tokens por conexão. Latência, erros, gargalos. Toda invocação de ferramenta registrada com identidade do chamador, entradas, saídas e timing. Um dashboard.

---

### Capacidades Principais

| Capacidade | Descrição |
|---|---|
| **Agentes** | Navegue, contrate e componha agentes de IA com atribuição de custos |
| **Conexões** | Roteie tráfego MCP por um endpoint governado com auth, proxy e token vault |
| **Projetos** | Organize agentes e conexões em torno de objetivos com UI adaptativa |
| **MCPs Virtuais** | Componha e exponha conjuntos governados de ferramentas como endpoints MCP |
| **Observabilidade** | Traces, custos, erros e latência por conexão |
| **Controle de Acesso** | RBAC — OAuth 2.1 + API keys por workspace/projeto |
| **Multi-tenancy** | Isolamento por workspace/projeto para config, credenciais, políticas e audit logs |
| **Event Bus** | Pub/sub entre conexões com entrega agendada/cron e garantias at-least-once |

### Opções de Deploy

| | |
|---|---|
| **Local** | `bunx decocms` — PostgreSQL embutido, totalmente privado |
| **Cloud** | [studio.decocms.com](https://studio.decocms.com) — controle projetos de qualquer navegador |
| **Equipe** | Conexões compartilhadas, papéis, atribuição de custos |
| **Enterprise** | Self-hosted na sua infraestrutura |
