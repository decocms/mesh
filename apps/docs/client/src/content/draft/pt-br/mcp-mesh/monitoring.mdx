## O que você consegue ver

O Monitoring do Mesh serve para operar MCP em produção:

- **latência** por tool/connection/agent
- **taxa de erro** e mensagens de falha
- **logs** das tool calls (com filtros)

## Como usar no dia a dia

- **Debug**: comece filtrando por agent/connection e depois por tool.
- **Performance**: compare latência por connection e identifique gargalos.
- **Governança**: use logs como trilha operacional para mudanças e incidentes.

## Dica

Se uma tool "some" do client, verifique:

- se a connection foi conectada corretamente
- se o agent está incluindo/excluindo a connection certa
- se a API key tem as permissões necessárias
