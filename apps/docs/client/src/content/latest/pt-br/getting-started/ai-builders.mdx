Construa e implante sua primeira aplicação de IA usando linguagem natural.

## Construa Seu Primeiro AI App

Crie um app que escreve poemas, salva-os e envia por email para sua equipe.

<StepGroup>
  <Step number={1} title="Explore a App Store">
    1. Faça login em [admin.decocms.com](https://admin.decocms.com)
    2. Navegue para sua organização e projeto
    3. Clique em **Apps** na barra lateral
    4. Navegue pelas integrações disponíveis ou abra "Store" para instalar novas
    5. Selecione o app Gmail da store
    6. Conecte o app e autorize sua conta  
    
    <Callout type="info">
      Apps são integrações pré-construídas que expõem tools. Eles servem como blocos de construção para suas aplicações de IA.
    </Callout>
 
 </Step>

  <Step number={2} title="Teste a integração na interface de chat">
    Abra **Nova Thread** na barra lateral e pergunte ao deco chat:
    
    ```
    Crie um poema sobre deco CMS, salve-o em um documento e envie por email para voce@suaempresa.com usando @SendEmail
    ```
    
    Observe o deco chat:
    - Criar o poema
    - Chamar `DECO_RESOURCE_DOCUMENT_CREATE` para salvá-lo
    - Chamar `SendEmail` da integração Gmail para enviá-lo
    
    <Callout type="tip">
      Use **menções @** para chamar tools específicos. Deco chat orquestra múltiplos tools automaticamente.
    </Callout>
  </Step>

  <Step number={3} title="Torne reutilizável">
    Transforme essa tarefa pontual em automação reutilizável:
    
    ### Crie um Agent
    
    Vá em **Agents** → **Novo Agent**:
    - Atribua ferramentas (criação de documentos, email)
    - Adicione instruções personalizadas para seu caso de uso
    - Pronto! Agora qualquer pessoa pode usar este agent
    
    ### Crie um Workflow
    
    Crie workflows a partir de:
    - **Workflows** → **Novo Workflow** e abra o deco chat OU
    - Pergunte ao deco chat usando `@workflow_management` em qualquer thread OU
    - Pergunte dentro de qualquer conversa com agent usando `@workflow_management`
    
    **Exemplo:**
    ```
    Crie um workflow para geração diária de poemas
    ```
    Itere naturalmente, como se estivesse atribuindo a tarefa a outra pessoa
    
    **Exemplo (específico):**
    ```
    Crie um workflow chamado "poem-mailer" que:
    1. Recebe entrada: tópico (string) e destinatários (string)
    2. Gera um poema sobre o tópico usando IA
    3. Salva em documentos com tags ["poema", "gerado-por-ia"]
    4. Envia email para destinatários toda manhã às 9h
    ```
    
    ### Crie uma View Customizada
    
    Construa interfaces para seus dados:
    ```
    Crie uma view chamada "galeria-de-poemas" que:
    - Mostra todos os poemas dos documentos
    - Exibe em grade de 3 colunas
    - Cada card mostra título, descrição, data de criação
    - Adicione barra de busca para filtrar por nome
    - Deixe bonito
    ```
    
    <Callout type="success">
      Em minutos você vai do chat à automação pronta. Agents cuidam das conversas, workflows das tarefas repetitivas, e views deixam tudo bonito.
    </Callout>
  </Step>
</StepGroup>

## Modos do Agent

Três modos especializados para diferentes estágios:

- **Explorer**, Descubra capacidades e tools
- **Planner**, Projete aplicações e escreva PRDs
- **Builder**, Implemente tools, workflows e views

Alterne entre modos através do seletor na interface de chat. Cada um tem acesso otimizado a tools.

<Callout type="info">
  Diferentes modos são otimizados para diferentes estágios: Explorer (descobrir tools), Planner (projetar apps) e Builder (implementar).
</Callout>

## O Que Você Pode Construir

### Ferramentas Internas
- Automação de suporte ao cliente
- Workflows de entrada e validação de dados
- Geração e distribuição de relatórios
- Sistemas de gerenciamento de inventário

### Apps de Produtividade
- Sumarizadores de notas de reunião
- Automatizadores de atribuição de tarefas
- Busca e recuperação de documentos
- Lembretes e follow-ups automatizados

### Processamento de Dados
- Importação e transformação de CSV
- Agregação de dados de API
- Verificações automatizadas de qualidade de dados
- Geração agendada de relatórios
