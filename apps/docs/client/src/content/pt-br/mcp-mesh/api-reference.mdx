## Endpoints MCP

### Management MCP

- `POST /mcp`

Esse endpoint expõe **management tools** via MCP (orgs, connections, gateways, API keys, monitoring, etc.).

### Proxy de connection

- `POST /mcp/:connectionId`

Faz proxy de requisições MCP para uma única connection upstream.

### Gateway (MCP virtual)

- `POST /mcp/gateway/:gatewayId`

Faz proxy de requisições MCP para um gateway (agregação + estratégia de exposição).

## Endpoints de autenticação (alto nível)

Os endpoints exatos podem variar conforme o provider/configuração, mas em geral o Mesh expõe rotas para:

- login/logout
- callback de OAuth
- sessão do usuário
