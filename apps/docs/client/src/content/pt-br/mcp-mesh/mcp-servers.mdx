## O que é uma Integração (no Mesh)?

No Mesh, uma **Integração** é uma **connection** para um MCP upstream (normalmente um endpoint MCP via HTTP).

Cada connection guarda:

- **URL do MCP**
- **credenciais/config** (quando necessário, armazenadas no vault)
- metadados para operação (nome, status, etc.)

## Quando usar vs Assistant

- Use **connection** para isolar e depurar um upstream específico.
- Use **Assistant** para expor um surface agregado/curado para os clients.

## Boas práticas

- Mantenha URLs e credenciais por ambiente (dev/staging/prod).
- Evite expor uma connection diretamente em produção; prefira um Assistant para governança.
